# Hero Background Images

This folder should contain the following hero background images:

1. **blinds-living-room.jpg** - Modern living room with venetian blinds
2. **shutters-bedroom.jpg** - Elegant bedroom with plantation shutters  
3. **office-blinds.jpg** - Professional office space with window treatments
4. **modern-kitchen-blinds.jpg** - Contemporary kitchen with roller blinds
5. **luxury-home-shutters.jpg** - Upscale home interior with custom shutters

## Image Requirements:
- **Resolution**: 1920x1080 or higher
- **Format**: JPG or WebP for best performance
- **Aspect Ratio**: 16:9 (landscape)
- **File Size**: Optimized for web (under 500KB each)
- **Content**: High-quality photos showcasing Venus Blinds products in real settings

## Fallback:
If images are not available, the component will show a gradient background until images are added.
